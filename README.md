## Getting Started

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Ahalya-KT/sec-lab-machine-task-react.git
   ```

2. Navigate to the project frontend directory:

   ```bash
   cd sec-lab-machine-task-react
   ```

   ```bash
   npm install
   ```

   ```bash
   npm run dev
   ```
